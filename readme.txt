== Description ==

Besitzt du noch kein XXLBoard Forum? [Hier kostenlos erstellen](http://wordpress.xxlboard.de/)

**Integriere dein Forum**

Dein XXLBoard Forum kannst du mit diesem Plugin einfach in eine Seite integrieren. Außerdem kannst du das Kommentarsystem durch dein Forum ersetzen!

Ferner existieren noch diverse Widgets zur besseren Integration deines Forums.

* **Blog Kommentare**: Ersetze das Kommentarsystem deines Blogs mit dem XXLBoard Kommentarsystem. Dies bedeutet, dass neue Artikel ein eigenes Thema in deinem Forum erhalten und die Kommentare entsprechend aufgenommen werden!

* **Forum einbetten**: Keine Style-Anpassungen notwendig. Du musst lediglich eine Seite zum Anzeigen des Boards auswählen.

* **Widgets hinzufügen**: Füge interessante Widgets zu deiner Sidebar hinzu: Aktuelle Diskussionen, Benutzer etc.

== Installation ==

1. Lade den Ordner `vanilla-forums` nach `/wp-content/plugins/`
2. Aktiviere das Plugin unter 'Plugins'
3. Konfiguriere das Plugin unter Einstellungen > XXLBoard

== Frequently Asked Questions ==

= Q. Was ist XXLBoard? =

XXLBoard bietet dir unter http://xxlboard.de kostenlos ein Forum an.

= Q. Warum sollte ich die Kommentarfunktion von XXLBoard verwenden? =

* Du kannst deine Kommentare in deinem Forum verwalten.
* Artikel werde auch im Forum verlinkt.
* Mehr Aufrufe: Blog-Benutzer sehen auch dein Forum (und umgekehrt).

= Q. Was ist der Unterschied zwischen XXLBoard und Disqus oder Intense Debate? =

Das Besondere ist, das deine Kommentare in ein Forum aufgenommen werden und somit noch übersichtlicher aufgerufen werden können!
